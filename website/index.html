Ett projekt i nätverksprogrammering

Som ett projektarbete i kursen Nätverksprogrammering är vi fyra studenter från olika civilingenjörsprogram som tillsammans har gjort ett program som låter användare spela det klassiska kortspelet Uno mot varandra från olika datorer via nätverk.
